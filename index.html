Calculadora Binaria

Introdueix un número decimal i et retornaré el seu valor en binari, hexadecimal i octal

Decimal, Binary, Hexadecimal, and Octal Conversion

            
# Decimal to Binary
num = int(input("Type a decimal number: "))
binary_num = bin(num)
print(f"Decimal number is: {num} and in Binary: {binary_num[2:]}")

# Binary to Hexadecimal
binary_string = binary_num
decimal_representation = int(binary_string, 2)
hexadecimal_string = hex(decimal_representation)
print("Hexadecimal value is:", hexadecimal_string.upper()[2:])

# Binary to Octal
bnum = binary_num
onum = int(bnum, 2)
onum = oct(onum)
print("Octal value is = ", onum[2:])
